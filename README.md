# Manufacturing-Operations-Analytics
Role: Business Analyst

Tools Used: Excel • SQL • Power BI • Tableau • PowerPoint

Domain: Manufacturing Operations Analytics

Objective: Provide end-to-end visibility on production, rejections, wastage trends, and departmental performance for leadership and factory managers.

## Project Overview

The Manufacturing Analytics Dashboard provides a consolidated view of daily production performance.
It enables stakeholders to track manufacturing quantity, rejected output, wastage %, machine-level inefficiencies, employee performance issues, and trends over time.

This solution helps in:

Identifying bottlenecks in production

Minimizing rejection and waste

Improving machine and employee efficiency

Enhancing department-level visibility

Supporting data-driven decision making

## Skills Demonstrated

Data Cleaning & Transformation (Power Query & Excel)

DAX Measures for KPIs

Building Interactive Power BI and Tableau Dashboards

Trend & Department-level analytics

Data storytelling & stakeholder reporting
